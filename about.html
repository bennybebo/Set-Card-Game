Rules

The objective of this game is simple, you have select a 'Set' of three cards from a deck of 12 cards laying on the table.

Each card has a unique version of the following four characteristics:

  • COLOR: Each card has one of three colors: red, green, or blue.
  • SYMBOL: Each card contains one of three symbols: ovals, squiggles, or diamonds.
  • NUMBER: Each card has either one, two, or three symbols.
  • SHADING: Each card has a shading of either filled, empty, or striped background.

Examples of sets:

  • COLOR: different on each card, SYMBOL: the same on each card (oval), NUMBER: the same on each (two), SHADING: the same on each card (solid)
  • COLOR: different on each card, SYMBOL: different on each card, NUMBER: different on each, SHADING: different on each card
  • COLOR: the same on each card (green), SYMBOL: the same on each card (diamond), NUMBER: different on each, SHADING: different on each card
